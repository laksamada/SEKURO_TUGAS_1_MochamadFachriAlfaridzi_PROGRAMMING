Halo, di video ini saya akan menjelaskan workspace ROS2 yang saya buat.

Workspace saya ada di folder `src/ros_soal_1/ros2_ws`.
Di dalam folder `src` ada package bernama `simple_chat`.
Package ini memakai `ament_python`.

Di package ini ada dua node.
Node pertama adalah publisher dengan nama `talker_node`.
Node kedua adalah subscriber dengan nama `listener_node`.

Publisher mengirim pesan ke topic `chat_topic` setiap 1 detik.
Pesan yang dikirim bertipe `std_msgs/msg/String`.
Contoh isi pesannya adalah `Halo dari publisher, pesan ke-1`, lalu terus bertambah.

Subscriber menerima pesan dari topic `chat_topic`.
Saat pesan diterima, subscriber akan menampilkan isi pesan ke terminal.

Proses build dimulai dengan masuk ke folder workspace.
Lalu saya aktifkan environment ROS2 dengan `micromamba activate ros_env`.
Setelah itu saya jalankan `colcon build`.
Sesudah build selesai, saya jalankan `source install/local_setup.bash`.

Lalu saya buka dua terminal.
Di terminal pertama saya jalankan `ros2 run simple_chat listener`.
Di terminal kedua saya jalankan `ros2 run simple_chat talker`.

Kalau saya cek `ros2 node list`, akan terlihat node `/talker_node` dan `/listener_node`.
Kalau saya cek `ros2 topic list`, akan ada topic `/chat_topic`.
Kalau saya cek `ros2 topic info /chat_topic`, tipe pesannya adalah `std_msgs/msg/String`.

Jadi alurnya sederhana:
publisher mengirim data ke topic, lalu subscriber membaca data dari topic itu.

Sekian penjelasan dari saya.
